Goji Boilerplate

goji-boilerpalate is a project skeleton for the Goji web framework that includes Twitter Bootstrap SASS, Font Awesome, Bower and Gulp for assets compilation and minification.

It also promotes seperation of concerns and app re-usability. This makes building middle and big size projects easier.
